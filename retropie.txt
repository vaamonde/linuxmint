O RetroPie permite que você transforme seu computador em uma máquina de jogos retrô, o projeto se baseou no Raspbian, EmulationStation, RetroArch e muitos outros, permitindo relembrar os jogos clássicos de Arcade e dos famosos Consoles Atari, Mega Driver, Super Nintendo etc, com o mínimo de configuração de hardware (EXEMPLO você pode utilizar uma placa do Raspberry Pi 2/3/4), você pode jogar utilizando o Teclado ou Joystick (EXEMPLO você pode usar joystick dos consoles Playstation 2/3/4, Xbox 360/One, Genérico, Arcade, etc) para instalar o RetroPie no Mint, temos várias etapas.

#Primeira Etapa: Verificação da versão do Linux Mint
sudo hostnamectl        #Verificando informações do sistema
sudo lsb_release -c     #Verificando o codinome do 
sudo uname -a           #Verificando a versão do Kernel

#Segunda Etapa: Atualização do Sistema Operacional Linux Mint
sudo apt update             #Atualização das listas do apt
sudo apt upgrade            #Atualização dos software
sudo apt dist-upgrade       #Atualização do firmaware e kernel
sudo apt autoremove         #Remoção dos softwares desnecessários
sudo apt autoclean          #Limpeza dos caches
sudo apt clean              #Limpeza dos caches

#Terceira Etapa: Habilitar a Arquitetura i386 e Repositório Universal
sudo dpkg --add-architecture i386		#Arquitetura i386
sudo add-apt-repository universe		#Repositório Universal
sudo apt update             			#Atualização das listas do apt

#Quarta Etapa: Instalação das Dependências do RetroPie
sudo apt install joystick jstest-gtk vim git

#Quinta Etapa: Clocando o Projeto do RetroPie do Github
git clone --depth=1 https://github.com/RetroPie/RetroPie-Setup.git

#Sexta Etapa: Executando a instalação do RetroPie
cd RetroPie-Setup				#Acessando o diretório clonado
sudo ./retropie_setup.sh		#Executando o script de instalação
